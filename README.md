## 先搞清楚：搬瓦工是什么

搬瓦工，英文名 BandwagonHost，是加拿大 IT7 Networks 公司旗下的 VPS 服务品牌。因为英文发音和"搬瓦工"的拼音相近，国内用户就一直这么叫。

VPS（Virtual Private Server，虚拟私有服务器）是一台放在海外数据中心的虚拟机，你拥有完整的 root 权限，装什么系统、跑什么程序、怎么配置都是你说了算。和你买一台本地服务器没有本质区别，只是它在机房里，通过网络连进去操作。

搬瓦工从 2004 年就开始做了。早年靠超低价的入门套餐打出名气，后来逐步把低价普通机器砍掉，转型做中高端 CN2 优化线路。现在在售的套餐，重点就是针对中国大陆访问做了线路优化——洛杉矶 CN2 GIA、香港 CN2 GIA、日本软银等，这些线路高峰期也相对稳定。

## IP 被封这件事怎么理解

搜"搬瓦工"经常会看到有人提 IP 被封。这是国内网络环境的问题，不是搬瓦工的问题，也不是"非法"的证明。

什么叫 IP 被封：是指你的 VPS IP 在国内某些运营商的网络层面被屏蔽，导致无法直接访问。这和商家是否合规、服务是否合法完全是两回事——是网络层的屏蔽策略，不是法律层面的处罚。

正规使用的情况下，IP 被封的概率很低。用过搬瓦工 CN2 GIA 套餐好几年、一直建站用的人，很多从没遇到过这个问题。IP 被封大多是因为被滥用——同一网段的其他用户干了坏事，波及到邻近 IP。

如果真的遇到 IP 被封，搬瓦工支持花 $8.79 换一个 IP，或者通过 KiwiVM 控制面板迁移到其他机房，换一个 IP 段重新开始。

## CN2 GT 和 CN2 GIA 的区别，以及为什么重要

这个问题值得单独说一下，因为很多人买错了套餐后觉得搬瓦工"不行"，其实是买到了 GT 而不是 GIA。

**CN2 GT（Global Transit）**：去程走普通线路，回程走 CN2。高峰期可能拥堵，速度不稳定，晚上偶尔会变慢。

**CN2 GIA（Global Internet Access）**：去程和回程全走 CN2 专线，高峰时段会绕开拥挤的骨干节点，延迟更低，丢包更少。洛杉矶到国内大约 130–160ms，日本软银大约 60–80ms，香港能到 30–60ms。

现在搬瓦工的 KVM 入门套餐走的其实是普通线路或 CN2 GT，CN2 GIA-E 才是真正的全程 GIA。如果你需要国内访问速度快，选 CN2 GIA-E，不要因为省那点钱买 KVM 然后对速度失望。
